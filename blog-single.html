Saya mencapai rank ini dengan berbagai macam cara yaitu dengan bermain solo, duo, trio, hingga party.

Saya dulu punya partyan namanya Red Wedding... dulu merupakan partyan paling aktif pemimpin squadnya bernama Rafiendra Rahman Pramudya cuman sekarang dia udah fokus ke pekerjaannya sebagai programmer di machine vision.

Jadi sekarang yang memegang squad adalah saya sendiri, karena saya diberi kepercayaan oleh beliau untuk menjaga squad tersebut.

Fun fact saya sangat suka sekali dengan game ini. Saya dulu pernah menjadi robot ml semasa SMP yang dimana saya setiap hari bermain game ini hingga lupa waktu sehingga nilai saya menjadi jelek. tetapi semenjak saya masuk SMK ini saya sudah jarang sekali bermain game ini.
